About

The WeightTracker helps you keep track of your weight.

It presents an intuitive interface which allows everyone to feel at home right from the start. The modern design and simple structure makes the app suitable for almost everyone. The WeightTracker is usable on all kind of devices, be they phones, tablets or desktops.

Running your own WeightTracker instance could also not be easier: The application has been built using Docker, which makes deploying it to any kind of host machine very easy.

The tool stores all your data using just three entities, as seen here: The center of the WeightTracker is the WeightEntry, which stores the weight of a single person on a specific day. There can be many such entities for each user, although each user may only have one for a specific day. Also, users have the ability to attach images to their weight entries. This allows one to visually see their progress over time.

The WeightTracker consists of the following features:

  • Users can create new accounts.
  • Users can sign in using their account.
  • Users can change the username of their account.
  • Users can change the password of their account.
  • Users can delete their account.
  • Users can save a single weight for a specific day.
  • Users can edit and delete any of their saved weights.
  • Users can attach images to any of their weight entries.

Weights are presented to the user in a chronological order, using a line chart. Clicking on the data points of the chart allows the user to edit the corresponding weight entry.
